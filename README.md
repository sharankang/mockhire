## **Mockhire is an AI-driven platform designed to help job seekers prepare for interviews and optimize their resumes. It leverages Google's Gemini AI to provide real-time feedback, simulate interview scenarios, and evaluate resumes against job descriptions.**

## Features

### 1. ATS Resume Evaluator

- Smart Analysis
- AI Feedback
- Validation

### 2. AI Interview Simulation

- Realistic Practice
- Dual Modes (Text and Video)
- Real-time Feedback
- Final Report

### 3. Interview Question Generator

- Tailored Questions

### 4. User Profile & History

- Dashboard
- Resume Storage
- Simulation History

### 5. Authentication

- Secure Access

## Tech Stack

### Frontend

* HTML5 & CSS3
* JavaScript (ES6+)
* Web APIs: SpeechRecognition (for Speech-to-Text), SpeechSynthesis (for Text-to-Speech), MediaDevices (for Webcam access)

### Backend

* Node.js & Express.js
* MongoDB & Mongoose
* Google Gemini API
* JWT (JSON Web Token)
* PDF.js
